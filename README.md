# Vue Project Portfolio
This application was built using Vue.js and version controlled using Git through Microsoft's Visual Studio Code IDE.
The primary purpose of this application is to illustrate my software engineering projects in a simple, intuitive web interface.
 
## Building the application
To build this application, you must run the following console command:

```
npm run build
```

### Further installation guidance
For a complete tutorial into Vue.js installation, please read their documentation at: https://vuejs.org/v2/guide/installation.html
